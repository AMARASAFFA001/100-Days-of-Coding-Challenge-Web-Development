HTML & CSS Basic Summary

HTML Summary

HTML (HyperText Markup Language) is used to defined our page content, structure and meaning. You dont use it for styling purposes. Use CSS for that instead!

  • HTML User "element" to describe ( annotate) content
  • HTML element typically have an opening tag, content and then closing tag
  • You can also have void (empty) element like images
  • You can aslo configure element with attributed
  • Theres a long list of available element but you will gain experience overtime, on worries.

Learn more about all available HTML on the MDN HTML element reference.

CSS Summary

CSS (Cascading Style Sheet) is used for styling your page content.

  • style are assigned via property-value pairs
  • You can assign styles via the "style" attribute
  • To avoid code dulplication, you typically use global styles ( via the "style" element) instead
  • Alternativelly you can work with external styleshit files which you " link" to
  • When working with CSS, concept like "inheritance", "specificity" and the "box model" should be understood.

Learn more about all available CSS properties and values on the MDN CSS Propertyreference.
